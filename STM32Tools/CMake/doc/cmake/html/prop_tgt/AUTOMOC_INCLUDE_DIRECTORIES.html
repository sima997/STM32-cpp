AUTOMOC_INCLUDE_DIRECTORIES¶

Added in version 4.1.

Specifies zero or more include directories for AUTOMOC to pass explicitly to the Qt Meta‑Object Compiler (moc) instead of automatically discovering a target's include directories.

When this property is set on a target, only the directories listed here will be used by AUTOMOC, and any other include paths will be ignored.

This property may contain generator expressions.

All directory paths in the final evaluated result must be absolute. If any non-absolute paths are present after generator expression evaluation, configuration will fail with an error.

See also the CMAKE_AUTOMOC_INCLUDE_DIRECTORIES variable, which can be used to initialize this property on all targets.

Example¶

add_library(myQtLib ...)
set_property(TARGET myQtLib PROPERTY AUTOMOC_INCLUDE_DIRECTORIES
  "${CMAKE_CURRENT_SOURCE_DIR}/include/myQtLib"
)
